## Description
My Diary App is an Android application developed using Kotlin in Android Studio.  
It allows users to create, edit, and manage their daily diary entries securely.

## Features
- User registration and login
- Add new diary entry
- Edit diary entry
- Delete diary entry
- View saved entries

## Technologies Used
- Kotlin
- Android Studio
- SQLite Database

## How to Run

2. Open in Android Studio
3. Run on emulator or Android device
